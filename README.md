# Portfolio Website

A Next.js portfolio website with no lint errors and a default homepage.

## Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

### Development

Start the development server:
```bash
npm run dev
```

The website will be available at `http://localhost:3000`

### Build

Build the project for production:
```bash
npm run build
```

### Start Production

Start the production server:
```bash
npm start
```

### Linting

Check for lint errors:
```bash
npm run lint
```

## Project Structure

- `pages/` - Next.js pages (homepage is at `pages/index.js`)
- `public/` - Static files
- `package.json` - Project dependencies and scripts
- `next.config.js` - Next.js configuration
- `.eslintrc.json` - ESLint configuration

## Customization

Edit the files in the `pages/` directory to modify your portfolio content and styling. The homepage is located at `pages/index.js`.
